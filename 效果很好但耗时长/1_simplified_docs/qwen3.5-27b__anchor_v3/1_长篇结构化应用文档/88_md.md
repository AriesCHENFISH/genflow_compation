# 一周冬季不囤肉轻食食谱

**核心原则**：慢碳水+优质蛋白+时令根茎菜；低温烹煮；日热量1200-1500kcal。

**周一**
早：燕麦南瓜粥 (燕麦30g+南瓜100g)+水煮蛋1个+凉拌菠菜(100g)
午：糙米饭(80g)+清蒸鲈鱼(150g)+白萝卜菌菇汤
晚：山药鸡丝羹(山药80g+鸡胸肉50g)+焯水西兰花(100g)

**周二**
早：紫薯泥(150g)+无糖豆浆+芹菜香干(芹菜50g+香干30g)
午：杂粮饭团(糙米40g+藜麦10g)+味噌豆腐(嫩豆腐100g+海带芽5g)
晚：魔芋结(200g)+羊肉萝卜汤(羊肉30g+白萝卜100g)

**周三**
早：全麦蔬菜饼(全麦粉40g+蛋1个+胡萝卜30g)+低脂奶200ml
午：蒸红薯(150g)+虾仁炒芦笋(虾仁80g+芦笋100g)
晚：银耳百合羹(干银耳5g+鲜百合30g)+凉拌木耳(50g)

**周四**
早：小米山药粥(小米30g+山药50g)+茶叶蛋1个+芝麻拌菠菜(100g)
午：荞麦面(60g)+日式冷汁+焯水秋葵(80g)
晚：番茄菌菇豆腐煲(番茄50g+金针菇30g+嫩豆腐100g)

**周五**
早：南瓜小米糊(南瓜100g+小米20g)+鹌鹑蛋3个+凉拌莴笋丝(50g)
午：杂豆饭(红豆20g+糙米40g)+葱烧鸡胸(鸡胸100g+大葱20g)
晚：韩式海带汤(干海带5g+牛肉片30g)+蒸胡萝卜块(100g)

**周六**
早：藜麦水果酸奶碗(熟藜麦30g+无糖酸奶150g+柚子肉50g)
午：姜母鸭(鸭胸80g+老姜20g)+清炒油麦菜(100g)
晚：莲藕排骨汤(莲藕50g+小排30g)+凉拌魔芋丝(100g)

**周日**
早：蔬菜豆腐脑(内酯豆腐150g+香菇丁20g)+蒸玉米段(100g)
午：三文鱼杂粮饭(三文鱼80g+糙米50g+牛油果30g)
晚：上汤娃娃菜(娃娃菜200g+皮蛋半颗)+蒸芋头(80g)

**执行要点**
餐前喝200ml温水或菜汤；橄榄油日限15g；19:00前进晚餐，餐后散步20min；加餐选原味杏仁10颗/柚子2瓣/无糖酸奶100g。
