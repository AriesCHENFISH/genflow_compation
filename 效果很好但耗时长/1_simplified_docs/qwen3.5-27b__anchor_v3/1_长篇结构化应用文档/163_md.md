# 工程技术规范
1. 支承系统对准中线，防骨架倾斜。门框门槛与人防底板贴合度≤30mm，超用反梁。涂层耐磨系数≥40。
2. 临空墙禁石砌/塑料卡，用直径≥12 固定止水螺杆，止水片 40X40X3。超压排气阀中心距≥600mm。
3. 中弦杆 m2-2N1: 2∠100×100×12, A2=38.64cm²。后浇带钢板厚度≥3mm，底板受拉钢筋搭接≤25%。
4. 电器设备高差：墙面≤5mm，室内≤10mm。焊缝饱满严密。水平筋过暗柱保保护层。
5. 严禁幕墙龙骨拆卸，需技术认可指导。灌注时间<首批混凝土初凝时间。
