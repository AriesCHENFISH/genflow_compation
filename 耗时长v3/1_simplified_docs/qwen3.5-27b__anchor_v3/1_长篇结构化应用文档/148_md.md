# 盘源性疼痛机制
1.化学：退变释放前列腺素等炎性因子，刺激神经。2.力学：纤维环撕裂致稳定性破坏，异常活动机械刺激。3.解剖：病变处纤维环、终板神经密度高，敏感性强。4.免疫：髓核内发现抗原抗体复合物及IgG沉积。

# 病理过程
炎症介质经裂隙达外层纤维环，致窦椎神经超敏；形成炎性肉芽组织条带区疼痛。相关文献：CT引导下造影诊断价值（《中国临床医学影像杂志》2007）。

# 治疗策略
## 非手术与手术
含微创、融合术。
## 微创技术
1.激光消融：X线/CT导引穿刺，气化减压回纳髓核，减少出血，缩短住院及恢复期。2.IDET（热疗）：热索沿纤维环加温，修复病灶。3.射频消融：加热突出物致蛋白凝固、减压回缩并修复纤维环。
## 人工椎间盘置换(ADR)
原理：恢复运动学及载荷特性，去除炎症源。
并发症：短期（骨折、脱位、感染、血管神经损伤）；远期（假体衰竭、下陷、邻节段病变）。
## 人工髓核置换(NPR)
文献：海涌等(2006)、金大地等(2002)。
适应症：非手术无效的慢性腰背痛；典型放射痛伴或不伴腰痛。
禁忌证：纤维环缺损、小关节增生、管狭窄、不稳、畸形及骨质疏松。
并发症：脱出、移位、膨胀不全、椎间盘炎、症状难缓解。
评价：保留节段活动，短期疗效好，长期数据尚缺。
