# 人教版五下语文生字描红字帖
**概况**：依教材编写，含全册生字。A4 打印，12 格/字，配课文序。
**生字映射**（每页6组）：
第1页：赵璧、召诺、怯瑟、拒诸、荆妒、忌曹。
第2页：督甘、鲁延、幔私、寨擂、呐援、丞擞。
第3页：绽扳、咚监、侄郎、皆敛、媳骚、宗怜。
第4页：帕脊、莞锦、姹嫣、暇颇、尼艇、叉艄。
第5页：翘雇、哗融、祭奠、旬募、熊炼、依呜。
**设计特点**：标准田字格定位；充足练习空间；教材同步选型；排版简洁无干扰。
**使用建议**：
1. 笔具：低年级先铅笔后钢笔；高年级直用钢笔/中性笔。
2. 方法：先观结构再按笔顺写，配合朗读记音义。
3. 频次：每周3-4页，循序渐进，忌贪多。
4. 场景：课堂/作业教学，家长听写检验。
**目标**：提升书写规范性与美观度，培养良好习惯，夯实语文基础。
