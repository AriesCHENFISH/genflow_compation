# 2025 秋冬家庭养生饮食计划
## 一、核心原则
1. **季节适配**：润燥选银耳/梨/山药（避寒凉）；御寒用羊肉/姜/桂圆（控量，羊周1次200g）。日均菜300-500g，深色占半。
2. **人群适配**：老人（软烂少盐糖≤5g）；儿童（日12种食材，趣味造型）；上班族（早餐备餐，晚餐清淡）。例：萝卜羊肉汤一锅三调（去油/加枣/正常）。
3. **安全烹饪**：肉海鲜熟透≥75℃；避生冷，凉拌加姜蒜当日食；生熟分开，洗手20秒烫菜板。

## 二、一日三餐食谱（3-4人）
### 早餐（≤20min）
1. **经典**：银耳百合小米粥 + 蔬菜鸡蛋饼。原理：养胃润燥。过敏者换蔬菜小馒头。
2. **快手**：南瓜山药粥 + 坚果馒头 + 温拌黄瓜。优势：健脾滋阴，无煎炒。

### 午餐（≤40min）
1. **家常**：萝卜羊肉汤 + 杂粮饭 + 清炒西兰花。原理：温补解腻，低升糖。忌羊肉可换鸡胸肉。
2. **清淡**：番茄金针菇豆腐汤 + 葱油饼 + 凉拌菠菜。优势：补钙开胃，暖身不腻。

### 晚餐（≤30min）
1. **经典**：山药莲子瘦肉粥 + 蒸南瓜 + 清炒油麦菜。原理：易消化，健脾蛋白。
2. **快手**：蔬菜鸡蛋面 + 凉拌木耳。优势：20min搞定，急救方案。

## 三、针对性食疗方
1. **秋燥咳**：川贝蒸梨（川贝粉+蜂蜜）。干咳连吃3天；黄痰加枣，白痰加姜。
2. **手脚凉**：当归生姜红枣茶（当归+姜+枣+红糖）。上午饮，感冒发烧禁服。
3. **脾胃弱**：山药茯苓粥。食欲不振/腹胀连喝7天，不加糖。

## 四、避坑指南
1. **忌大补**：全员参茸易上火，改用温和食材少量添粥汤。
2. **忌多肉**：儿日肉量=掌心，选瘦肉蒸煮，防便秘。
3. **忌无菜**：老人大便软切碎入粥面，保底300g。
4. **忌冰水**：全家用35℃温水，防痉挛腹泻。
5. **忌久存**：剩菜限24h，保鲜盒装，次日彻底加热（≥75℃）。
