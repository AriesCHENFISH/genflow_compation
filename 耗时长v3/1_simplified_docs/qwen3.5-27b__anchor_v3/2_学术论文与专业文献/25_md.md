# 珠海洪湾淤泥特性研究
**来源**：中山大学学报（自然科学版）2024(3)
**目的**：探明珠海洪湾地区原状淤泥工程特性。
**方法**：取ZK-1~3孔（6.1~15.7m深）18个原状样，开展基本物性、渗透、一维固结及固结不排水三轴试验。
**关键数据**：
1. **物理特性**：含水率ω=54.7%~64.9%，密度ρ=1.02~1.11g/cm³，塑性指数Ip=18.7~24.4；渗透系数k=3.68~5.86×10⁻⁷cm/s，透水性差。
2. **压缩特性**：属高压缩性土。压缩系数av=1.269~1.618MPa⁻¹，模量Es=1.449~1.919MPa。随深度增加，av减小，Es增大。
3. **抗剪强度**：黏聚力c=10.10~20.84kPa，内摩擦角φ=7.28°~12.09°。呈现应变硬化；随深度增加，φ微增，c整体降低。
**结论**：该区域淤泥强度高差、变形大、透水性弱；参数随深度显著变化，需结合具体深度评估工程风险。
