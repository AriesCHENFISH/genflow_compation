英语朗读宝:15分钟/天,4周,前2单元单词+课文。
语文朗读宝:正版下册教材,随年级切换。

**优惠方案**
1. 试听课:跨年特惠99元,2科,1小时/科。
2. 正价课:1月1日前缴费不涨价,购≤50节享9.5折。

**三个举一反三**
1. 宏观(书):通读目录,画思维导图,理清章节关系。
2. 中观(单元):精读核心章节,梳理概念、定理、案例,建知识网络。
3. 微观(知识点):深挖例题,变条件、逆向提问、类比迁移、模拟出题。

**学习计划制定**
细化每日时段,如晨语文、午数学、晚英语,劳逸结合。可采用番茄工作法:25分钟学习+5分钟休息,4循环后休息15-30分钟。

**高三寒假备考**
1. 锚定目标:评估试卷错题,制定时间表与目标,定期限时套题训练。
2. 调整心态:运动、倾诉。
3. 各科复习:
   -语文:限时练习(语用+信息类45分钟,文言+诗歌35分钟),即时改悟,必要拓展。
   -数学:多练,总结。
   -英语:早起阅读,背单词,每日刷题总结,读1-2篇文章,听歌,写春节作文。
   -日语:完成作业,规划每日学习。
   -历史:坚持看书做题,体验春节历史,看新闻。
   -政治:完成试卷,看新闻联播和春晚,查漏补缺。
   -地理:看《航拍中国》,观察春节天气风俗,每日学习20分钟。

**习惯叠加法**
将新习惯“锚定”于现有习惯后。如早餐后阅读15分钟。大脑回路:提示→行动→奖赏。利用锚点降低启动门槛。
寒假锚点示例:早起穿衣后→口算1页;早餐喝牛奶后→背1单词;午饭后→听英语15分钟;看动画片后→练字1行;晚饭前→运动;洗澡后→读书10分钟。任务宜小。

**21天寒假大通关**
1. 设计:游戏化三阶关卡(基础→提升→拓展),即时贴纸奖励。
2. 内容:数学错题归因,英语听力二维码,语文阅读理解五步法,附学霸笔记模板。
3. 优势:21天计划表可视化,期末诊断卷定位薄弱,预习用漫画导图。
4. 使用指南:
   -第1-7天:晨做诊断卷,下午专项训练,晚看学霸笔记。
   -第8-14天:穿插趣味模块,画下学期导图。
   -第15-21天:预习章节闯关测验,整理错题本。

**三阶学习法**
1. 预复习3:7法则:30%时间建知识路标,70%攻错题盲区。单元雷达图标识薄弱,推送变式题。
2. 每日学习胶囊:晨15分钟漫画,午30分钟AI组题,晚20分钟实践任务(如翻译价签、计算水电费)。
3. 游戏化激励:解锁知识宝石,兑换错题消除卡。
4. 隐藏价值:
   -知识地图:扉页脉络树。
   -错题诊断:类型,拓展题,口诀。
   -跨学科项目:春节任务包(如统计压岁钱)。
